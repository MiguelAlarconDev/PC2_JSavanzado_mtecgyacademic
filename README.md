# MTECGYacademic

Aplicación web tipo e-commerce desarrollada con Angular para el curso de Javascript Avanzado.

## Tecnologías utilizadas

- Angular
- TypeScript
- HTML
- CSS
- Bootstrap
- JSON Server
- LocalStorage

## Funcionalidades principales

- Navbar reutilizable
- Footer reutilizable
- Arquitectura basada en componentes
- Home
- Productos dinámicos desde JSON Server
- Ofertas
- Tienda
- Contacto
- Mi Cuenta
- Login simulado
- Dashboard protegido
- Página 404 personalizada
- Carrito de compras
- Buscador y filtro por categorías
- Uso de directivas, pipes y data binding

## Instalación

```bash
npm install
```

## Ejecutar Angular

```bash
ng serve
```

Abrir en el navegador:

```text
http://localhost:4200
```

## Ejecutar JSON Server

```bash
npm run server
```

API local:

```text
http://localhost:3000/productos
```
